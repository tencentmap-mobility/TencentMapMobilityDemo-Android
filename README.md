# 出行SDK接入文档（安卓）

## 概述

出行SDK是针对于出行场景开发的多个SDK，包括:周边车辆SDK,推荐上车点SDK，出行检索SDK和司乘同显SDK。每个SDK功能独立、接入方便，开发者可以根据自己的需求进行选择。

## 接口文档地址
https://tencentmap-mobility.github.io/
